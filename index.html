Apex-1 Launch Coaster — Real-Time Control System

Company: SunRail Dynamics  |  Platform: ESP32 / FreeRTOS  |  Simulator: Wokwi

Project Description

SunRail Dynamics is a Central Florida theme park control-systems integrator. This proof-of-concept validates that an ESP32 running FreeRTOS can meet all hard real-time deadlines for a hydraulic launch coaster — where a missed brake-sensor deadline could mean launching into engaged brakes, causing catastrophic mechanical failure and rider injury.

The system uses five FreeRTOS tasks plus a hardware ISR, coordinated through a mutex, counting semaphore, binary semaphore, event group, and telemetry queue. Serial proof timestamps confirm every hard deadline was met across 18+ seconds of continuous operation with zero misses.

Deadline Proof

Serial timestamps confirm BrakePoll fires every 50 ms (start=450 → start=500 → start=550, Δ=50 ms) with dur=0 execution time. Watchdog fires every 100 ms. Zero MISS lines observed across 18+ seconds. E-Stop ISR response confirmed by estop flag flipping within one watchdog cycle (100 ms) of button press.
